Como enviar um edital

Fluxo em 3 passos
  1. Clique em Enviar PDF e abra a pasta inputs/ no GitHub.
  2. Faça upload do PDF do edital nessa pasta e confirme o commit.
  3. O GitHub Actions processa o arquivo e o GitHub Pages publica a nova análise.

Como a análise funciona

Pipeline de processamento
  1. O workflow extrai o texto do PDF com pdftotext e usa OCR quando a página parece escaneada.
  2. O script procura padrões de requisito, classifica cada trecho por tipo e registra página, seção e confiança.
  3. O site mostra o resultado bruto, os requisitos filtrados e uma comparação entre editais para identificar itens únicos, compartilhados e duplicatas exatas.
